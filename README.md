# 🎯 Why SentinelAI?

Every day millions of users face:

❌ Cyberbullying

❌ Hate Speech

❌ Toxic Comments

❌ Threats & Harassment

Traditional keyword filters fail.

SentinelAI leverages **Natural Language Processing + Machine Learning + Ensemble Intelligence** to identify harmful content in real-time with high accuracy.

# 🏗️ System Architecture

```text
Social Media Text
        │
        ▼
 NLP Preprocessing
(Cleaning + Tokenization +
 Stopword Removal +
 Lemmatization)
        │
        ▼
 TF-IDF Vectorization
        │
        ▼
 SMOTE Balancing
        │
        ▼
 ML Models
 ├─ Logistic Regression
 ├─ Linear SVM
 ├─ Random Forest
 └─ Naive Bayes
        │
        ▼
 Ensemble Decision Engine
        │
        ▼
 Toxicity Prediction
```

# 🧠 AI Pipeline

### Stage 1 — Data Processing

* URL Removal
* HTML Cleaning
* Tokenization
* Lemmatization
* Slang Expansion

### Stage 2 — Feature Engineering

* TF-IDF
* Unigrams
* Bigrams

### Stage 3 — Class Balancing

* SMOTE Oversampling

### Stage 4 — Model Training

* Logistic Regression
* Linear SVM
* Random Forest
* Naive Bayes

### Stage 5 — Real-Time Moderation

* Ensemble Voting
* Confidence Scoring
* Category Detection
