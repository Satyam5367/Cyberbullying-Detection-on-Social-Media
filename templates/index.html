ABOUT THIS PROJECT
Group 8 — NLP Machine Learning Project · Cyberbullying Detection on Social Media

🎯 Project Overview

CyberGuard AI is a machine learning system to detect and classify cyberbullying across social media platforms using an ensemble of 4 classical ML models.

Dataset: Synthetic data (10,000 samples) mimicking the Kaggle Toxic Comment Dataset and Twitter Hate Speech dataset. Distributed across 7 categories with realistic class imbalance.

Stack: Python · scikit-learn · TF-IDF · NumPy · Pandas · Matplotlib · Seaborn · Flask · Chart.js

⚙️ Full NLP Pipeline

  • 1

    Text Preprocessing

    Lowercase, URL/mention removal, slang expansion (tbh→honestly), stopword filtering, punctuation stripping, elongation normalization

  • 2

    Feature Engineering

    Char count, word count, CAPS ratio, exclamation count, unique word ratio, avg word length, aggression pattern regex hits

  • 3

    TF-IDF Vectorization

    Unigrams + bigrams, 10K max features, sublinear TF scaling, min_df=2, max_df=0.9, unicode accent stripping

  • 4

    Ensemble Prediction

    Majority voting across 4 models — content flagged if ≥2 models vote bullying. Best model: Linear SVM (F1=0.9145)
